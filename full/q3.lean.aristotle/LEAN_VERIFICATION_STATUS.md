### 1. S_K_small (Geometric Series Decay)
```
theorem S_K_small (K t η : ℝ) (hK : K ≥ 1) (hη : η > 0) (ht : t ≤ t_min K η) :
    S_K K t ≤ η
```
For small enough t, the geometric series S_K can be made arbitrarily small.

### 2. node_spacing (Node Separation)
```
theorem node_spacing (K : ℝ) (hK : K ≥ 1) :
    ∀ (n₁ n₂ : ℕ), n₁ ∈ Nodes K → n₂ ∈ Nodes K → n₁ < n₂ →
      xi_n n₂ - xi_n n₁ ≥ delta_K K
```
Adjacent spectral nodes are separated by at least δ_K.

### 3. off_diag_exp_sum_bound (Off-Diagonal Sum)
```
theorem off_diag_exp_sum_bound (K t : ℝ) (hK : K ≥ 1) (ht : t > 0) (i : Nodes K) :
    ∑ j : Nodes K, (if j ≠ i then exp(-(ξᵢ - ξⱼ)²/(4t)) else 0) ≤ S_K K t
```
The off-diagonal Gaussian sum is bounded by the geometric series.

### 4. W_sum_is_finite (Weight Sum Bound)
```
theorem W_sum_is_finite (K : ℝ) (hK : K > 0) : ∃ B, W_sum K ≤ B
```
The sum of prime weights over active nodes is finite.

### 5. Q_Lipschitz_local (Lipschitz Continuity)
```
theorem Q_Lipschitz_local (K : ℝ) (hK : K > 0) :
    ∃ L > 0, ∀ Φ₁ ∈ W_K K, ∀ Φ₂ ∈ W_K K,
      |Q Φ₁ - Q Φ₂| ≤ L * sup{|Φ₁(x) - Φ₂(x)| : x ∈ [-K,K]}
```
The Q functional is Lipschitz on W_K.

### 6. Q_nonneg (Atom Positivity)
```
theorem Q_nonneg (K : ℝ) (hK : 1 ≤ K) (a_star : ℝ → ℝ) ...
    (h_A3 : A3BridgeProperty K a_star T_arch)
    (h_RKHS : RKHSContractionProperty K) :
  ∀ g ∈ AtomCone K, 0 ≤ Q a_star g
```
If A3 Bridge and RKHS contraction hold, Q is nonnegative on the atom cone.

### 7. A1_density_main (Density Theorem)
```
theorem A1_density (K : ℝ) (hK : K > 0) (Φ : ℝ → ℝ) (hΦ : Φ ∈ W_K K) (ε : ℝ) (hε : ε > 0) :
    ∃ g ∈ AtomCone_K K, sup|Φ(x) - g(x)| < ε
```
Fejér×heat atoms are dense in W_K (approximation theory).

# SCALING FIX — Критическая проблема нормировки

## УБИЙЦА ДОКАЗАТЕЛЬСТВА

**Было в Dossier:**
> "Q(Φ) ≥ 1.125 для всех Φ в конусе"

**Почему это НЕВОЗМОЖНО:**
```
Если Φ допустима, то cΦ тоже допустима.
Q(cΦ) = c · Q(Φ)
При c → 0: Q(cΦ) → 0

⇒ Нельзя иметь uniform lower bound > 0 для ВСЕХ Φ!
```

---

## ПРАВИЛЬНАЯ ЦЕЛЬ

**Вариант A (без нормировки):**
$$Q(\Phi) \ge 0 \quad \forall \Phi \in \mathcal{W}$$
Это именно то, что нужно для Weil criterion.

**Вариант B (с нормировкой):**
$$Q(\Phi) \ge c_* > 0 \quad \text{при } \|\Phi\|_\infty = 1 \text{ или } \Phi(0) = 1$$
Без явной нормировки — это математика-убийца.

# Q3 paper workspace

Структура:
- `tex/` — LaTeX-исходники статьи и Makefile.
- `notes/` — чеклисты, assumption-audit, план работ (`assumption_audit.md` и т. д.).
- `ops/` — скрипты/утилиты для сборки и проверок.
- `figs/` — рисунки и визуализации, если понадобятся.

Все новые материалы под публикацию складываем сюда, чтобы не плодить хаос по всему репозиторию.

```
f(n) = Λ(n) · χ₄(n)

- χ₄(n) = 0   если n чётное

**Тождество A:**
```

## AFM СТРУКТУРА (Доказано)

**Теорема:** Для twin prime (p, p+2) где p > 2:
```
χ₄(p) · χ₄(p+2) = -1
```

**Доказательство:**
- Если p ≡ 1 (mod 4): тогда p+2 ≡ 3 (mod 4), произведение = (+1)·(-1) = -1
- Если p ≡ 3 (mod 4): тогда p+2 ≡ 1 (mod 4), произведение = (-1)·(+1) = -1

Это **antiferromagnetic** структура — ВСЕ twins имеют одинаковый знак! ∎

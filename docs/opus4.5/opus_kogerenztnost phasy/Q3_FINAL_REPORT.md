# Q3 PROJECT: ИТОГОВЫЙ ОТЧЁТ

## Резюме

Q3 — операторный подход к гипотезе Римана через критерий Вейля.

**Главный результат:** Численная верификация критерия Вейля для RH и GRH.

---

## Фаза 1: Гипотеза Римана (RH)

### Конструкция

$$H = T_A - T_P$$

где:
- $T_A$ — архимедов оператор (Toeplitz матрица)
- $T_P$ — праймовый оператор (сумма rank-1 матриц)

### Результаты

| K | Простых | min λ(H) | Статус |
|---|---------|----------|--------|
| 0.5 | 9 | -7.49×10⁻¹⁵ | ✅ |
| 1.0 | 99 | -1.55×10⁻¹³ | ✅ |
| 1.5 | 1,479 | -1.60×10⁻¹⁴ | ✅ |
| 2.0 | 24,976 | ~10⁻¹³ | ✅ |
| 2.5 | ~10⁵ | ~10⁻¹³ | ✅ |

**Вывод:** Критерий Вейля H ≥ 0 выполняется на всех проверенных компактах.

---

## Фаза 2: Близнецы (TPC)

### Подход 1: Heat Trace

$$\text{Tr}(e^{-\tau A^{(2)}} V) \sim \tau^{-\alpha}$$

- Результат: α ≈ 0.75 на промежуточных τ
- **Проблема:** α — артефакт, не истинная асимптотика
- **Проблема:** V почти одномерная (rank ≈ 3)
- **Статус:** ❌ Не работает

### Подход 2: χ₄-симметрия

**Открытие:** χ₄(p) × χ₄(p+2) = -1 для ВСЕХ близнецов!

- **Проблема:** Heat регуляризация убивает χ₄-симметрию
- При t = t_min: дисбаланс ~99%
- **Статус:** ❌ Конфликт с регуляризацией

---

## Фаза 3: GRH (Обобщённая RH)

### Конструкция

$$H_\chi = T_A^\chi - T_P^\chi$$

для характеров Дирихле χ.

### Результаты

| L-функция | min λ(H) | Статус |
|-----------|----------|--------|
| ζ(s) | -1.43×10⁻¹³ | ✅ |
| L(s, χ₄) | -1.77×10⁻¹³ | ✅ |
| L(s, χ₃) | -1.39×10⁻¹³ | ✅ |

**Вывод:** GRH для χ₄ и χ₃ численно подтверждена.

---

## Что доказано (численно)

### ✅ Сильные результаты:

1. H ≥ 0 для RH на компактах K ≤ 2.5
2. H_χ ≥ 0 для GRH (χ₄, χ₃) на K ≤ 2.0
3. Tr(T_P_TWINS) растёт с K (β ≈ 0.37)

### ⚠️ Слабые результаты:

4. Heat trace α ≈ 0.75 — артефакт
5. H_twins ≥ 0 — не различает конечное/бесконечное

### ❌ Не работает:

6. χ₄-симметрия убивается регуляризацией
7. Спектр V не накапливается (низкий ранг)

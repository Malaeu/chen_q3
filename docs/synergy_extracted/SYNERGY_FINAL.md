## КЛЮЧЕВОЕ ОТКРЫТИЕ

**ЗОЛОТОЙ МОСТ между подходами:**

$$\chi_4(n) \cdot e(n/4) = i \quad \text{для всех нечётных } n$$

Это связывает:
- **AFM structure** (χ₄ даёт protection)
- **Резонанс при α=1/4** (F_{1/4} даёт пик)
- **Некоммутативность** ([F_{1/4}, U_2] ≠ 0 даёт forcing)

## ЛЕММА PB (Parity Breaking) — ЕДИНСТВЕННЫЙ GAP

Нужно доказать:

**Пункт 1 (энергетика):**
$$\text{Bell}(X) \geq c \cdot X^{1/2+\delta}$$

Следует из некоммутативности и Robertson uncertainty.

**Пункт 2 (распаковка):**
$$\text{Bell}(X) = \alpha \cdot S_2(X) + O(\sqrt{X} \log^2 X)$$

Следует из AFM: T_χ₄ = -S₂.
